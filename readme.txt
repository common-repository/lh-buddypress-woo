Move WooCommerce My Account area to BuddyPress profile

== Description ==

Make all customer relevant data accessible from one place. Add pages from other WooCommerce extensions right into your BuddyPress member profiles.

== Changelog ==

**1.00 August 08, 2020**  
Initial release.

== Installation ==

1. Upload the entire `lh-buddypress-woocommerce` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. There are no settings, the plugin just works
